### MDI程序，多窗口管理
这是一个基于Qt6的MDI（多文档界面）应用程序示例。该程序允许用户在同一窗口中打开和管理多个子窗口，每个子窗口可以包含不同的文档。

#### 功能
- 新建文档
- 打开现有文档
- ~~保存文档~~
- 剪切、复制、粘贴文本
- 设置文本字体
- 关闭所有子窗口
- 支持MDI模式和标签页模式切换
- 子窗口级联和平铺布局
- 子窗口高亮颜色特征显示

#### 依赖
- Qt 6.8或更高版本

#### 编译和运行
1. 确保已安装Qt6开发环境。
2. 已添加CMakePresets.json确保任何支持cmake3.19以上版本的ide可以直接打开项目
3. 编译并运行项目。
